# Recipes
# Каталог за готварски рецепти

## Описание
"Каталог за готварски рецепти" е ASP.NET Core MVC приложение, което позволява на потребителите да създават, преглеждат, редактират и изтриват готварски рецепти. Приложението поддържа управление на категории, съставки и детайлна информация за всяка рецепта.

## Функционалности
- **CRUD операции** за рецепти, категории и съставки
- **Свързване на рецепти с категории**
- **Дефиниране на съставки за всяка рецепта**
- **Качване и показване на изображения към рецептите**

## Технологии
- **ASP.NET Core MVC** – за уеб приложението
- **Entity Framework Core** – за управление на базата данни
- **Microsoft SQL Server** – за съхранение на данни

## Структура на базата данни
Приложението използва следните основни модели:

### Recipe (Рецепта)
- `RecipeId` – Уникален идентификатор
- `Title` – Заглавие на рецептата
- `Description` – Кратко описание
- `CookingTime` – Време за готвене
- `Instructions` – Инструкции за приготвяне
- `CategoryId` – Връзка към категорията
- `ImagePath` – Път до изображение
- `RecipeIngredients` – Колекция от съставки

### Category (Категория)
- `CategoryId` – Уникален идентификатор
- `Name` – Име на категорията
- `Recipes` – Списък с рецепти в категорията

### Ingredient (Съставка)
- `IngredientId` – Уникален идентификатор
- `Name` – Име на съставката
- `RecipeIngredients` – Рецепти, които съдържат тази съставка

### RecipeIngredient (Съставки в рецепта)
- `RecipeIngredientId` – Уникален идентификатор
- `RecipeId` – Връзка към рецепта
- `IngredientId` – Връзка към съставка
- `Quantity` – Количество

## Инсталация и стартиране
1. **Клониране на хранилището:**
   ```bash
   git clone https://github.com/your-repository-url.git
   cd your-repository-folder
   ```
2. **Конфигуриране на базата данни:**
   - Промени `appsettings.json`, за да добавиш връзка към базата данни.
   ```json
   "ConnectionStrings": {
     "DefaultConnection": "Server=.;Database=RecipesDb;Trusted_Connection=True;"
   }
   ```
3. **Изпълнение на миграции:**
   ```bash
   dotnet ef database update
   ```
4. **Стартиране на приложението:**
   ```bash
   dotnet run
   ```
5. **Отваряне в браузър:**
   ```
   http://localhost:5000
   ```
